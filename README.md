Инновационный, ролевой AI-бот в Telegram от лица Махиро Ояма (Mahiro Oyama) из аниме *"Onimai: I'm Now Your Sister!"*. 
Бот оснащён собственной экономикой, долгосрочной памятью, системой распознавания изображений, продвинутой админ-панелью и полноценным Web-Dashboard.

## ✨ Основные возможности (Features)

* **🧠 Продвинутый AI & Память:** Использование Mistral AI (Pixtral-12B) с поддержкой долгосрочной и краткосрочной памяти. Бот запоминает интересы пользователей!
* **👀 Vision (Распознавание фото):** Бот умеет смотреть на отправленные ему фотографии и комментировать их в образе Махиро.
* **🎤 Генерация голоса (TTS):** Бот умеет генерировать голосовые сообщения.
* **💰 Экономика & Подарки:** Система опыта (XP), доверия (Trust) и валюты (MahiroCoins). Заработанные монеты можно тратить на подарки Махиро!
* **🎮 Интерактив:** Настраиваемые викторины (Quiz), таблица лидеров и уникальные достижения (Achievements).
* **👑 Web Dashboard 2.0:** Управление пользователями, просмотр статистики и изменение AI-настроек ("Prompt Tuning") прямо из браузера через FastAPI.
* **🛡 Мощная Модерация:** Система анти-спама, "Служба Поддержки" с системой тикетов, встроенные команды бана, белый/черный списки.
* **💸 Интеграция Telegram Stars:** Поддержка платежей через Telegram Stars для покупки MahiroCoins и VIP-статусов.

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/YOUR_USERNAME/mahiro-bot.git
cd mahiro-bot
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка окружения (.env)
Скопируйте пример конфига и заполните свои данные:
```bash
cp .env.example .env
```
Укажите внутри `.env` свой токен Telegram, API ключ Mistral и ID администратора.

### 4. Запуск бота
```bash
python main.py
```
После запуска бот начнет работу, а Web-Дашборд будет доступен по адресу `http://localhost:8000`.

## 📂 Структура проекта
- `ai/` - Логика работы с Mistral AI (текст и зрение)
- `bot/` - Основной функционал бота, клавиатуры, мидлвари и обработчики
- `database/` - Модели и SQLite репозиторий
- `media/` - Обработка изображений (графики) и генерация голоса (TTS)
- `memory/` - Долгосрочная и краткосрочная память
- `utils/` - Антиспам, триггеры, достижения, бэкапы
- `web/` - FastAPI дашборд с REST API для управления ботом
- `data/`, `logs/`, `cache/` - Директории (авто-создаются) для хранения локальных данных.
