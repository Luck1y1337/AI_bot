## 📖 О Проекте

**Mahiro-bot** — это не просто Telegram-помощник, это полноценный виртуальный собеседник от лица Махиро Ояма (Mahiro Oyama) из аниме *"Onimai: I'm Now Your Sister!"*. Бот обладает собственным настроением, памятью (запоминает ваши интересы), системой достижений, а также невероятно проработанной экономикой и социальными механиками.

Проект создан с упором на интерактивность, геймификацию и удобный интерфейс с использованием **Inline-кнопок** (aiogram 3) вместо скучных текстовых команд.

## ✨ Полный Функционал Проекта (Features)

### 🧠 Искусственный Интеллект и Память
- **Ролевое общение:** Бот строго придерживается характера Махиро.
- **Vision (Зрение):** Отправьте боту фотографию, и он проанализирует её содержимое, выдав уместный ролевой комментарий.
- **Двухуровневая память:** 
  - *Краткосрочная:* Поддержание контекста диалога в реальном времени.
  - *Долгосрочная:* Извлечение фактов и интересов пользователя и запись их в локальную базу данных, чтобы бот мог ссылаться на них спустя месяцы общения.
- **Динамическое настроение и доверие:** Ваши действия (подарки, общение) влияют на шкалу настроения бота и уровень доверия (Trust) к вам.

### 💰 Продвинутая Экономика (Economy 2.0)
- **Валюта и Опыт:** Система `MahiroCoins` 🪙 и `XP` ✨.
- **Бизнесы (Пассивный доход):** 
  - Покупка бизнесов (например, Студия Манги, Игровая Компания).
  - Сбор пассивной прибыли в зависимости от прошедшего времени (до 12 часов оффлайн-накопления).
- **Магазин (Shop):** Покупка уникальных **Титулов**, которые отображаются в вашем профиле и Таблице Лидеров, а также виртуальных подарков для бота.
- **Банковская Система:** 
  - *Депозиты:* Вкладывайте коины под 5% годовых (доход капает каждый день).
  - *Кредиты:* Берите займы, но помните о процентах!
- **HammaPay (Переводы):** Безопасная система перевода средств между игроками (защита от переводов самому себе и отрицательных сумм).

### 🎮 Игры и Социальные Механики
- **Казино (Coinflip PvP):** Бросайте вызов другим игрокам на монеты! Интерактивный процесс с броском кубика `🎲`.
- **Викторины (Quiz):** Инлайн-викторины на знание лора. За правильные ответы начисляются XP и монеты.
- **Система Браков 💍:** Сделайте предложение другому игроку! Женатые пары получают перманентный бафф: **+10% ко всем доходам с бизнесов**.
- **Репутация (HammaRep):** Раз в сутки вы можете повысить репутацию (+50 XP) любому активному игроку на сервере.
- **Таблица Лидеров:** Показывает Топ-10 игроков с их купленными Титулами и статистикой.
- **Система Достижений:** Уникальные ачивки за игровую активность (например, "Первый миллион", "Мастер Викторин").

### 👑 Администрирование и Модерация
- **Полностью In-App Админ Панель:** Управляйте ботом прямо через Telegram (кнопка `👑 Админ Панель`).
- **Управление доступом:**
  - *Whitelist (Белый список):* Система приватного доступа с рассмотрением заявок.
  - *Blacklist (Черный список):* Мгновенный бан нарушителей прямо в базе данных.
- **Управление балансом:** Принудительная выдача коинов и XP через удобное меню выбора пользователей.
- **Рассылка (Broadcast):** Массовая отправка сообщений (с фото) всем пользователям бота.
- **Система Промокодов:** Создавайте промокоды с лимитом использований на коины и XP (`/addpromo`).

### 🌐 Web Dashboard (Веб-панель)
- Полноценный интерфейс на **FastAPI + Jinja2**.
- **Real-time Аналитика:** Графики активности (генерация через `matplotlib`), количество пользователей, состояние памяти.
- **Управление Настройками:** Включение/выключение режима Maintenance (Тех. обслуживание), Whitelist, изменение системных промптов нейросети "на лету" без перезагрузки бота.

## 🚀 Установка и Запуск

> [!IMPORTANT]
> Для корректной работы бота необходимы API ключи от **Telegram** (через @BotFather) и **Mistral AI**.

### 1. Подготовка
Склонируйте репозиторий:
```bash
git clone https://github.com/YOUR_USERNAME/mahiro-bot.git
cd mahiro-bot
```

### 2. Зависимости
Убедитесь, что у вас установлен **Python 3.10+**.
```bash
pip install -r requirements.txt
```

### 3. Настройка Конфигурации
Создайте файл `.env` в корневой папке проекта на основе примера:
```bash
cp .env.example .env
```
В файле `.env` обязательно заполните:
```env
BOT_TOKEN="ваш_токен_телеграм"
MISTRAL_API_KEY="ваш_ключ_mistral"
ADMIN_USER_IDS="123456789,987654321" # ID администраторов через запятую
```

### 4. Запуск
```bash
python main.py
```
*При первом запуске бот автоматически сгенерирует локальную базу данных `data/bot_database.db` и все необходимые системные папки.*

Веб-дашборд по умолчанию будет доступен по адресу: `http://localhost:8000`

## 📂 Структура Архитектуры (Clean Architecture)

Кодовая база разбита на логические модули для обеспечения легкой масштабируемости и чистоты кода:

```text
mahiro_bot/
├── ai/             # Ядро интеграции с Mistral/Pixtral, генерация текста и анализ фото
├── bot/            
│   ├── handlers/   # Роутеры Aiogram (Экономика, Игры, Админка, Основной чат)
│   ├── fsm/        # Состояния конечных автоматов (States) для интерактивных меню
│   ├── keyboards/  # Сборка Inline-клавиатур
│   └── middlewares/# Системы перехвата (Антиспам, Вайтлист)
├── database/       # ORM-подобный Repository Pattern (aiosqlite)
├── media/          # Работа с медиафайлами, генерация графиков аналитики
├── memory/         # Менеджер Памяти: долгосрочное хранилище интересов пользователя
├── web/            # FastAPI сервер, шаблоны и статика для дашборда
├── config/         # Загрузка и валидация .env переменных (Pydantic-like)
├── data/           # (Автоматически) Локальная база SQLite
└── main.py         # Основной входной файл (Асинхронный запуск Бота и FastAPI)
```

## 📚 Основные Команды для Пользователя

Помимо общения на естественном языке, пользователи могут использовать команды:
* `/start` — Начать общение / подать заявку в приватного бота.
* `/menu` — Открыть главное интерактивное меню (Игры, Экономика, Профиль).
* `/stats` — Посмотреть свой профиль, баланс, титулы и бизнесы.
* `/leaderboard` — Посмотреть топ-10 игроков на сервере.
* `/promo <code>` — Активировать промокод от администратора.
* `/ticket` — Написать в службу поддержки.

*(Большинство функций вынесено в удобные **Inline-кнопки**, доступные через `/menu`!)*
